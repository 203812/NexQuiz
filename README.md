# 🚀 NexQuiz

**NexQuiz** is a modern multiplayer quiz platform designed to make learning competitive, engaging, and fun.  
Compete with friends or classmates in real-time quiz battles and climb the leaderboard.

---

## 🎯 Features

- 🎮 Real-time multiplayer quiz battles
- 🏆 Live leaderboard system
- ⚡ Fast and responsive UI
- 🎨 Modern design with clean animations
- 📱 Mobile-friendly interface
- 🧠 Custom quiz creation (planned)
- 🔐 Secure game sessions

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- (Optional future backend: Firebase / Supabase / Node.js)
